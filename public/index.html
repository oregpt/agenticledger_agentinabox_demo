Catalyst Canton - Version 1.11 Release Notes

What's New

  • Validator Backup and Restore - Configure database backup schedules through Catalyst and restore validators with ease. Read more →
  • Identity Backups Scheduling - Configure the identity backup flow through a schedule. These backups are now also visible in the Catalyst UI. Read more →
  • Configure External DB - Create validators configured for external database services (e.g., RDS, Cloud SQL).
  • User Roles for DAML Versions - Added roles for Participant users in new DAML versions on the Edit User UI.
  • Enhanced Monitoring Dashboard - New metrics and visualizations for validator health and network status.
  • Improved API Documentation - Comprehensive OpenAPI specs with interactive examples. Read more →

Bug Fixes

  • Connection Timeout Issues - Fixed intermittent connection timeouts during high-load scenarios.
  • UI Rendering - Resolved display issues on mobile devices and smaller screens.
  • Certificate Validation - Improved TLS certificate handling for custom CA configurations.

Known Issues

  • Large file uploads (>100MB) may timeout on slower connections. Workaround: Use chunked upload API.
  • Safari browser may show intermittent WebSocket disconnections. Fix scheduled for 1.12.
